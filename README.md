# Automated CI/CD Pipeline for Java Train Ticket Reservation Application

## Overview

This project demonstrates an end-to-end CI/CD pipeline for a Java-based Train Ticket Reservation Application using GitHub Actions, Docker, Amazon Elastic Container Registry (ECR), Apache2, and Amazon EC2 (Ubuntu).

## Technologies

- Java
- Maven
- JSP & Servlets
- HTML

- GitHub
- GitHub Actions
- Docker
- Amazon ECR
- Amazon EC2
- Apache2
- Ubuntu Linux
